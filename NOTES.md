# Git課題 メモ

## 課題1

### git initの前後でディレクトリの中身に何が増えたか
.git という隠しフォルダが作られた。
Gitの管理情報（履歴・設定など）がすべてここに入っている。

### ステージとは何か
コミットする予定のファイルを一時的に置く場所。
git add でファイルをステージに上げる。

### コミットとは何か
ステージの内容をスナップショットとしてGitの歴史に記録すること。

## 課題2

### "+"と"-"はそれぞれ何を表しているか
+ は追加された行、- は削除された行を表す。

### コミットした後にもう一度差分を表示すると何が表示されるか
何も表示されない。
コミット後はワーキングツリーとリポジトリの内容が一致するため、差分がゼロになるから。
